Variable - это именованная область памяти, содержимое которой 
может изменяться в процессе выполнения программы.

				type name;
type - тип переменной.

1. Логические типы: bool (Boolean)
	true	1
	false	0
2. Символьные типы: char (Character - Символ)
	ASCII
3. Числовые типы
	unsigned - могут хранить лишь положительные целые числа
	  signed - могут хранить как положительные, так и отрицательные целые числа

	  short		2B
	  long		4B
	  int		2B-4B
	  long long 8B
